2023 ARE YOU READY TO CODE?

HELLO

Imagine uma competição assim: Várias pessoas desenvolvedoras front-end precisam codificar uma página em apenas 15 minutos... Até aí o desafio já é enorme, concorda? Agora adicione o fato de que eleas não podem ver o resultado da página no browser, somente o código fonte, usando apenas HTML e CSS: Esse é o Code in the Dark!

Um total de 16 pessoas irão competir em 4 baterias. Em cada round, 4 competidores(as) se enfrentam. Quem vencer se classifica pra uma grande final. Sua missão é produzir o HTML e CSS de um layout. Você não irá conhecer o layout - sem previews neste jogo my friend - só irá receber uma imagem como guia para o que tem de ser produzido no início de cada bateria.

Sem Photoshop!

Os participantes de cada bateria serão convidados para o palco principal, e sentarão cada um em um computador. Todo mundo terá o mesmo editor aberto (ele pode ser testado aqui) em cada computador. Não será permitido sair deste editor até o final de cada rodada. Todo seu código estará sendo replicado em um monitor externo voltado para o público, que irá acompanhar o que os competidores estiverem fazendo.

Nossa equipe vai disparar o cronômetro e serão 15 minutos para superar seus limites e apresentar o melhor trabalho. Ao final nossa equipe irá abrir todos os layouts em um navegador para início da votação do público. Simples, certo? Venha se divertir com a gente.
